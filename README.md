# BetterWeb
### Developers:
Submit your websites to get feedbacks from other developers/users.

### Users:
Help make the web a better place by sharing your thoughts.

## Tech stack:
* Express
* MongoDB
* React
* Node js
* Redux
* Socket.io
* Local Passport
